Postfix evaluation function

Your Data Structures and Algorithms teacher has given you a bunch of cryptic-looking strings of numbers and math operators she calls postfix expressions. You're supposed to evaluate those expressions. Like any self-respecting computer science student, you decide to write a program and let the computer do the work for you.

Write a function evaluatePostfix that takes a postfix expression as a string parameter. The function should return the value of the evaluated expression.
