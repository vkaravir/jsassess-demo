Augmenting basic objects with collect

You want to perform a certain operation on all items in an array and get the results into an array. You realize that the pluck method you added in the previous exercise does not fill this need. So, you decide to add a collect method to the Array prototype which is a more general version of the pluck. The function should take a function as a parameter and return an array where the values are the return values of the given function.

Examples:

["abc", "longer string", "23"].collect(function(item) { return item.charAt(0);});
//-> ["a", "l", "2"]

[ [2, 3, 1], [4, 5, 1] ].collect(function(item) {
    var i, sum = 0;
	for (i = 0; i < item.length; i++) {
	   sum += item[i];
	}
	return sum / i;
});
//-> [2, 3.3333333333333335]
