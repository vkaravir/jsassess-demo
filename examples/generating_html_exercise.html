Module for generating HTML in JavaScript

Generating HTML from JavaScript is a common task. Still, it's tedious concatenation of strings at best, and ends up in unreadable and unmaintainable code. To ease your pain, you decide to write a piece of code to generate HTML through function calls.

You name the module h and decide that you need to generate div, p, ul, ol, and li elements. Thus, your h module has functions for each of these elements. Requirements for the functions:

  • Each function takes an arbitrary number of child elements as string arguments.
  • The first argument can be an object. In that case, the properties in the object should become attributes of the element in the HTML
  • The function should return the generated HTML as strings.

Also, take note that you should hide the internal implementation of your code and provide an API with only the functions specified.

Example:

h.div({ 'id': 'mydiv' }, h.p("TODO list"), 
      h.ul(h.li("Write tests"),
           h.li({ 'class': 'current' }, "Write code"),
           h.li("Test code"),
           h.li("Start over")));

should generate the following HTML. Note, that the white-space and indentation are for readability, and are not required in the exercise.

<div id="mydiv">
  <p>TODO list</p>
  <ul>
    <li>Write tests</li>
    <li class="current">Write code</li>
    <li>Test code</li>
    <li>Start over</li>
  </ul>
</div>
