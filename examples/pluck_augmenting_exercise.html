Augmenting basic objects with pluck

Often you find yourself fetching the same property from all objects in an array. To help you work, add a function pluck to the Array prototype which does just that. The function should take a property name as a parameter and return an array where only the given property for each object is.

Examples:

["abc", "longer string", ""].pluck("length");
//-> [3, 13, 0]

[ {'id': 23}, {'id': 15}, {} ].pluck("id");
//-> [23, 15, undefined]
